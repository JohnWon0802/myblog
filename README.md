# JohnWon0802 Portfolio

A modern, performance-optimized personal portfolio website built with semantic HTML and responsive CSS.

🔗 Live Site  
https://johnwon0802.github.io/myblog/

---

## 📌 Project Overview

This project was built to demonstrate:

- Responsive layout design
- Accessibility-first development
- Performance optimization techniques
- Clean UI with interactive neon effects
- Deployment workflow using Git & GitHub Pages

The goal was not just to design a website, but to build a production-ready static site with measurable quality metrics.

---

## 🚀 Key Features

- Responsive typography using `clamp()`
- Custom neon hover interaction
- WebP image optimization (2MB → 100KB compression)
- Semantic structure with `<main>` landmark
- Accessibility improvements (ARIA, contrast adjustments)
- Lighthouse performance tuning
- Deployed via GitHub Pages

---

## ⚡ Performance Optimization

### Image Optimization
- Converted PNG assets to WebP
- Reduced hero logo size from 2MB to 100KB
- Improved LCP stability

### Accessibility Improvements
- Added `<main>` landmark
- Improved color contrast ratios
- Added ARIA labels where necessary
- Structured heading hierarchy
